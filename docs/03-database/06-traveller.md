# Traveller

**Document ID:** TC-DB-006  
**Version:** 0.1  
**Status:** Stable

## Purpose

The Traveller entity represents the owner of the application.

## Attributes

- Traveller ID
- Name
- Traveller Profile
- Preferences
- Home Country
- Language

## Rules

- One Traveller can own multiple Trips.
- One Traveller Profile is reused across all Trips.

> One traveller. Many journeys.
