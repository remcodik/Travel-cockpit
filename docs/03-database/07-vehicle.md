# Vehicle

**Document ID:** TC-DB-007  
**Version:** 0.1  
**Status:** Stable

## Purpose

Vehicle information supports routing and energy planning.

## Attributes

- Vehicle ID
- Name
- Type (EV, Petrol, Diesel, Hybrid)
- Consumption
- Battery or Tank Capacity
- Preferred Charging/Fuel Settings

## Rules

- A Trip uses one active Vehicle.
- Vehicle data influences routing and AI recommendations.

> The vehicle adapts the journey, not the architecture.
