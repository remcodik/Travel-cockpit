# Current Context

**Document ID:** TC-PROD-007  
**Version:** 0.1  
**Status:** Stable

## Purpose

Current Context represents the information that is relevant for the traveller at this moment.

It allows Travel Cockpit to personalise the interface and AI suggestions without requiring additional user input.

## Current Context Contains

- Active Trip
- Active Accommodation
- Current Location
- Current Day
- Traveller Profile
- Trip Profile
- Vehicle
- Weather
- Planned Activities
- Completed Activities

## Usage

Current Context is used by:

- Dashboard
- Maps
- Discover
- AI
- Planning
- Weather
- Energy Management

## Principles

Current Context is dynamic and continuously updated.

It supports the traveller but never changes planning automatically.

## Guiding Principle

> Show what matters now.
